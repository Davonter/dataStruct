# 如何轻松写出正确的链表代码

1. 理解指针或引用的含义

2. 警惕指针丢失或内存泄漏

3. 利用哨兵简化实现难度

    带头链表
    
4. 重点留意边界条件处理

    > 1. 链表为空时，代码是否能够正常工作。
    > 2. 链表包含一个节点时，代码是否能够正常工作。
    > 3. 链表包含两个节点时，代码是否能够正常工作。
    > 4. 代码在处理头节点和尾节点时，能否正常工作。

5. 举例画图，辅助思考

6. 多写多练，没有捷径


练习题：

1. 单链表反转

2. 链表中环的检测

3. 两个有序链表的合并

4. 删除链表倒数第N个节点

5. 求链表的中间节点
